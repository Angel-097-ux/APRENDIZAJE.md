# Mis Aprendizajes en Desarrollo Web

En este proyecto de práctica he aplicado los siguientes conceptos:

### 1. Formularios en HTML
* **¿Para qué sirve?**: Permite capturar datos del usuario (nombre, email, mensaje).
* **Concepto clave**: Aprendí a usar etiquetas `<label>` vinculadas con el `id` de los `<input>` para mejorar la accesibilidad.

### 2. Estructura Semántica
* **¿Para qué sirve?**: Utilizo etiquetas como `<section>`, `<footer>` y `<h2>` para que el navegador y los buscadores entiendan qué parte de la página es cada cosa.

### 3. CSS Grid (visto en tu código como `grid-proyectos`)
* **¿Para qué sirve?**: Sirve para crear diseños de cuadrícula de forma mucho más fácil y ordenada que los métodos antiguos.

### 4. Atributos de Accesibilidad (Aria-label)
* **¿Para qué sirve?**: Veo que usé `aria-label` para que las personas que usan lectores de pantalla puedan navegar mejor mi formulario.
